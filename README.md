## ✨ Overview

**Lumina 3D** is a visually captivating, interactive web experience built to celebrate the legacy, passion, and indomitable spirit of FC Barcelona ("Més Que Un Club"). Guided by an immersive ambient soundtrack and smooth animations, users explore a 3D gallery of legends.

## 🚀 Features

- **Immersive 3D Gallery**: Interactive columns and cards rendered using [Three.js](https://threejs.org/).
- **Cinematic Transitions**: Dynamic visual effects including white flashes and golden hints.
- **Responsive Animations**: Elements interact fluidly, adjusting perfectly to different screens.
- **Narrative Journey**: Features a scrollytelling-style layout and a beautifully styled "love letter" to the club.
- **Audio Integration**: Plays synced background music to complete the emotional atmosphere.

## 📁 Repository Structure

```text
/
├── index.html       # The entry point containing the core DOM structure
├── css/
│   ├── base.css     # Design system and typography
│   ├── ui.css       # Interface components
│   └── gallery.css  # 3D gallery styling
└── js/
    ├── main.js      # Main entry point and orchestration
    ├── three-scene.js # Three.js rendering logic
    └── gallery.js   # Gallery interaction logic
```

## 🛠️ Usage

This project runs natively in the browser without any build tools!

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/multiscene-3d-tribute.git
   ```
2. Navigate to the folder:
   ```bash
   cd multiscene-3d-tribute
   ```
3. Open `index.html` in your favorite modern browser, or use an extension like **Live Server** in VS Code.
